# Personal Expense Tracker (CLI)

# A simple command-line expense tracker built with Python. Log your spending, view summaries, and persist data between sessions.

## Features

- Add expenses (amount, category, description)
- View all expenses in a table
- Total spending
- Category summary
- Saves data to JSON and loads on restart

## Setup

1. Create & activate venv
2. Install dependencies:
   pip install tabulate

## Run

python src/main.py
