🚀 Custom Tokenizer + Gemini API Integration

A Python-based project that integrates Google Gemini API with Hugging Face Tokenizer to:

Generate AI responses using Gemini

Tokenize text using AutoTokenizer

Display tokens with color visualization

Show token IDs and token counts

📁 Project Structure
tokenizer-project/
│
├── main.py # Entry point
├── tokenizer.py # Tokenization logic
├── gemini_api.py # Gemini API integration
├── config.py # Configuration (API keys, models)
└── requirements.txt # Dependencies
