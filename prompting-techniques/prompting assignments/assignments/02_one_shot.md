# Assignment 02 — One-Shot Prompting

## Task

Write a Python script that uses one-shot prompting to convert **casual WhatsApp-style messages into formal business emails**.

Provide one example pair in the prompt, then test it on two new messages:

- "please send me the report asap, boss is asking"

## Rules

- Exactly one example must appear in the prompt — no more
- The example must show both the informal input and the formal output
- The script must loop over both test messages and print results for each

## What will be checked

- Exactly one example is present in the prompt
- Both test messages are processed and printed
- The output reads as a proper formal email, not a one-liner
- Standard PROMPT / RESPONSE print format is used
