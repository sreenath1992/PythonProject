# Assignment 12 — Prompt Engineering vs Prompt Tuning vs Instruction Tuning

## Task
This is a concept and reflection assignment. No classification or extraction needed.

**Part A — Ask the model:**
Send a prompt asking Claude to explain all three concepts specifically for
a Python developer who has used the Anthropic API for one week.

**Part B — Your own note:**
After the model response, print your own hardcoded summary block (not from the API)
that answers these three questions in plain language:

1. Which of the three will you use in this course?
2. Which requires actual ML training infrastructure?
3. What is the one thing you should always try before considering fine-tuning?

## Rules
- Part A must be a real API call — not a hardcoded string
- Part B must be your own words printed after the response — not generated by the model
- Part B must answer all three questions explicitly

## What will be checked
- An API call is made for Part A
- Part B is a hardcoded print block (not from the model)
- All three questions in Part B are answered
- The distinction between "what you do" vs "what AI labs do" is clear in Part B
